STRUCTURE

1.Matches up to quarterfinals will be played as a best of 3 formats, thereafter as a best of 5.
2.Each game will be first to 11 points.
3.Umpire’s decision final.
4.Teams must report 30 minutes before their match.

Boys:
Team Size: Minimum - 3 players
Maximum - 5 players

Girls:
Team Size: Minimum - 3 players
Maximum - 4 players

Matches Format:
1. All matches are knockout matches for boys. In case the number of girls teams are not more than 8, the girls' event may have few Round-Robin matches. Otherwise, girls' events will also have knockout matches.
2. Standard team event format (A-B-C Vs X-Y-Z).
3. Each set up to quarterfinals will be best of three.
4. Quarter-final onwards each set will be best of five.
Note: Only one boys’ team entry and one girls’ team entry are allowed per college.

Mixed-Doubles:
Event Matches Format:
1. All matches are knockout matches.
2. Each set up to quarterfinals will be best of three.
3. Quarter-final onwards each set will be best of five.
Note: The boy and Girl representing a team should be from the same college. 3 mixed double entries are allowed per college.
